# Revival Strength — Josh Lee's Coaching Website

## Project Overview
Multi-page static website for Revival Strength, a fitness coaching business by Josh Lee (Denver-based, 7+ years experience). Dark, cinematic, premium gym aesthetic with gold accents.

## Tech Stack
- Pure HTML/CSS/JS — no frameworks, no build tools
- Google Fonts: Bebas Neue, DM Serif Display, DM Sans
- All images currently base64 embedded (to be extracted later)

## Design System
- Colors: Gold `#C8853A`, Dark BG `#091520`, Card BG `#0D1F2D`, Mid `#1A3345`, Text `#F5F0E8`, Muted `#8FA5B4`
- Typography: Bebas Neue (headlines), DM Serif Display italic (accents), DM Sans (body)
- Border radius: 2-4px (sharp/confident), gold borders at low opacity
- Animations: fade-up on scroll via IntersectionObserver

## Pages
- `index.html` — Landing page (hero, pillars, about, process, nutrition, testimonials, CTA, lead magnet popup)
- `offerings.html` — Coaching offerings (3 tiers, how it works, FAQ, CTA)

## Planned Features
- Stripe Payment Links for checkout (no backend)
- Calendly embed for booking consultations
- Photo gallery + YouTube embeds
- Contact form with auto-email (Formspree or EmailJS)
- Email capture popup connected to Mailchimp/ConvertKit
